# Competitors

## Competitor ranking

Returns a paginated list of competitors ranked by AI visibility.

<CodeGroup>

```bash cURL
curl -X GET "https://app.finseo.ai/api/v1/projects/{projectId}/competitors?timeframe=30d&limit=10&sortBy=mentions" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
response = requests.get(
    f"https://app.finseo.ai/api/v1/projects/{project_id}/competitors",
    headers=headers,
    params={"timeframe": "30d", "limit": 10, "sortBy": "mentions"}
)
competitors = response.json()["data"]["competitors"]

for i, comp in enumerate(competitors, 1):
    brand = " (you)" if comp["isUserBrand"] else ""
    print(f"{i}. {comp['name']}{brand} — {comp['totalMentions']} mentions")
```

```javascript Node.js
const res = await fetch(
  `https://app.finseo.ai/api/v1/projects/${projectId}/competitors?limit=10`,
  { headers: { Authorization: "Bearer YOUR_API_KEY" } }
);
const { data } = await res.json();
data.competitors.forEach((c, i) =>
  console.log(`${i + 1}. ${c.name} — ${c.totalMentions} mentions`)
);
```

</CodeGroup>

<ParamField query="timeframe" type="string" default="30d">Time window</ParamField>
<ParamField query="search" type="string">Search competitor name or domain</ParamField>
<ParamField query="sortBy" type="string" default="mentions">`visibility`, `mentions`, `position`, `sentiment`, `name`</ParamField>
<ParamField query="sortDir" type="string" default="desc">`asc` or `desc`</ParamField>
<ParamField query="page" type="integer" default="1">Page number</ParamField>
<ParamField query="limit" type="integer" default="50">Results per page</ParamField>
