## Competitor ranking

```bash
GET /api/v1/projects/{projectId}/competitors
```

Returns paginated competitor visibility ranking.

<ParamField query="timeframe" type="string" default="30d">Time window</ParamField>
<ParamField query="model" type="string">AI model filter</ParamField>
<ParamField query="language" type="string">Language filter</ParamField>
<ParamField query="tags" type="string">Tag filter</ParamField>
<ParamField query="search" type="string">Search competitor name or domain</ParamField>
<ParamField query="sortBy" type="string" default="mentions">Sort by: `visibility`, `mentions`, `position`, `sentiment`, `name`</ParamField>
<ParamField query="sortDir" type="string" default="desc">`asc` or `desc`</ParamField>
<ParamField query="page" type="integer" default="1">Page number</ParamField>
<ParamField query="limit" type="integer" default="50">Results per page</ParamField>

**Each competitor includes:** `id`, `name`, `domain`, `isUserBrand`, `isStarred`, `totalMentions`, `visibleResults`, `averagePosition`, `sentimentScore`, `promptCount`
