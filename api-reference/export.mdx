## Bulk export

```bash
GET /api/v1/projects/{projectId}/export
```

Exports all prompts with full results data. Requires `export` scope on the API key.

<ParamField query="timeframe" type="string" default="30d">Time window for results</ParamField>

**Response includes:**
- `prompts` - Array of prompts, each with:
  - Prompt metadata (id, text, domain, brand, language, tags, models)
  - `results` - Full array of tracking results with model, visibility, mentions, position, fan-out queries
- `exportedAt` - Timestamp of export
- `timeframeDays` - Timeframe used

Up to 1000 prompts per export. Use for data pipelines, custom dashboards, or backup.
