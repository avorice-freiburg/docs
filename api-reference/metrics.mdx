# Metrics

## Daily KPIs

Returns current vs previous period metrics with changes.

<CodeGroup>

```bash cURL
curl -X GET "https://app.finseo.ai/api/v1/projects/{projectId}/metrics?timeframe=30d" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
response = requests.get(
    f"https://app.finseo.ai/api/v1/projects/{project_id}/metrics",
    headers=headers,
    params={"timeframe": "30d"}
)
metrics = response.json()["data"]

print(f"Visibility: {metrics['current']['visibilityRate']}%")
print(f"Change: {metrics['changes']['visibilityRateChange']:+.1f}%")
print(f"Mentions: {metrics['current']['mentions']}")
```

```javascript Node.js
const res = await fetch(
  `https://app.finseo.ai/api/v1/projects/${projectId}/metrics?timeframe=30d`,
  { headers: { Authorization: "Bearer YOUR_API_KEY" } }
);
const { data } = await res.json();
console.log(`Visibility: ${data.current.visibilityRate}%`);
console.log(`Change: ${data.changes.visibilityRateChange}%`);
```

</CodeGroup>

<ParamField query="timeframe" type="string" default="30d">Time window</ParamField>
<ParamField query="model" type="string">AI model filter</ParamField>
<ParamField query="language" type="string">Language filter</ParamField>
<ParamField query="tags" type="string">Tag filter</ParamField>

## Time series

Returns daily data points for building charts.

<CodeGroup>

```bash cURL
curl -X GET "https://app.finseo.ai/api/v1/projects/{projectId}/metrics/timeseries?timeframe=30d" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
response = requests.get(
    f"https://app.finseo.ai/api/v1/projects/{project_id}/metrics/timeseries",
    headers=headers,
    params={"timeframe": "30d", "model": "chatgpt"}
)
chart_data = response.json()["data"]

for day in chart_data:
    print(f"{day['date']}: {day['visible']} visible, {day['mentions']} mentions")
```

</CodeGroup>

<ParamField query="timeframe" type="string" default="30d">Time window</ParamField>
<ParamField query="startDate" type="string">Custom start date (YYYY-MM-DD)</ParamField>
<ParamField query="endDate" type="string">Custom end date (YYYY-MM-DD)</ParamField>
<ParamField query="model" type="string">AI model filter</ParamField>
