# API Reference

Base URL: `https://app.finseo.ai/api/v1`

## Endpoints

### Projects

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/projects` | List all projects |
| GET | `/projects/:id` | Get project details |
| POST | `/projects` | Create a project |
| PUT | `/projects/:id` | Update a project |

### Prompts

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/projects/:id/prompts` | List prompts with metrics |
| POST | `/projects/:id/prompts` | Add a new prompt |

### Metrics

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/projects/:id/metrics` | Daily KPIs with period comparison |
| GET | `/projects/:id/metrics/timeseries` | Time series chart data |

### Competitors

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/projects/:id/competitors` | Competitor visibility ranking |

### Sources

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/projects/:id/sources` | Citation source ranking |

### Tags

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/projects/:id/tags` | List tags |
| POST | `/projects/:id/tags` | Create and apply a tag |

### Export

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/projects/:id/export` | Bulk export all data |

## Common query parameters

Most GET endpoints accept these parameters:

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `timeframe` | string | `30d` | Time window (e.g. `7d`, `30d`, `90d`) |
| `model` | string | all | Filter by AI model (`chatgpt`, `claude`, `perplexity`) |
| `language` | string | all | Filter by language code (`en`, `de`, `fr`) |
| `tags` | string | all | Comma-separated tag names or JSON array |
| `page` | number | 1 | Page number for pagination |
| `limit` | number | 50 | Results per page (max 500) |
