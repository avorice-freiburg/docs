# Projects

## List projects

<CodeGroup>

```bash cURL
curl -X GET https://app.finseo.ai/api/v1/projects \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

headers = {"Authorization": "Bearer YOUR_API_KEY"}
response = requests.get("https://app.finseo.ai/api/v1/projects", headers=headers)
projects = response.json()["data"]

for project in projects:
    print(f"{project['name']} - {project['domain']}")
```

```javascript Node.js
const res = await fetch("https://app.finseo.ai/api/v1/projects", {
  headers: { Authorization: "Bearer YOUR_API_KEY" },
});
const { data: projects } = await res.json();
projects.forEach(p => console.log(p.name, p.domain));
```

## Get project

```bash
GET /api/v1/projects/{projectId}
```

<ParamField path="projectId" type="string" required>The project ID</ParamField>

## Create project

Requires `write` scope.

<CodeGroup>

```bash cURL
curl -X POST https://app.finseo.ai/api/v1/projects \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "New Brand", "websiteUrl": "https://newbrand.com"}'
```

```python Python
response = requests.post(
    "https://app.finseo.ai/api/v1/projects",
    headers=headers,
    json={"name": "New Brand", "websiteUrl": "https://newbrand.com"}
)
```

</CodeGroup>

<ParamField body="name" type="string" required>Project name</ParamField>
<ParamField body="websiteUrl" type="string">Website URL</ParamField>
<ParamField body="domain" type="string">Domain (auto-extracted from URL if not provided)</ParamField>
<ParamField body="description" type="string">Project description (max 500 chars)</ParamField>

## Update project

Requires `write` scope.

```bash
PUT /api/v1/projects/{projectId}
```
