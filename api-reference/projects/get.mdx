---
title: Get Project
api: GET /api/v1/projects/{projectId}
---

Retrieves a single project by ID, including settings used for AI visibility tracking.

<ParamField path="projectId" type="string" required>
  MongoDB-style project identifier.
</ParamField>

  Bearer token (`sk_live_...`). Requires `read` scope.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://app.finseo.ai/api/v1/projects/67a1c2d3e4f5061728394a5b \
  --header 'Authorization: Bearer sk_live_xxxxxxxx'
```

```python Python
import requests

project_id = "67a1c2d3e4f5061728394a5b"
r = requests.get(
    f"https://app.finseo.ai/api/v1/projects/{project_id}",
    headers={"Authorization": "Bearer sk_live_xxxxxxxx"},
)
print(r.json()["data"])
```

</RequestExample>
