---
title: List Projects
api: GET /api/v1/projects
---

Returns a list of projects your API key can access. Projects represent tracked brands or domains in Finseo for AI visibility across ChatGPT, Claude, Perplexity, and other models.

  Bearer token (`sk_live_...`). Requires `read` scope.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://app.finseo.ai/api/v1/projects \
  --header 'Authorization: Bearer sk_live_xxxxxxxx'
```

```python Python
import requests

response = requests.get(
    "https://app.finseo.ai/api/v1/projects",
    headers={"Authorization": "Bearer sk_live_xxxxxxxx"},
)
projects = response.json()["data"]
for p in projects:
    print(p["name"], p["id"])
```

</RequestExample>
