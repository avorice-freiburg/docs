## Source ranking

```bash
GET /api/v1/projects/{projectId}/sources
```

Returns paginated list of sources cited in AI responses.

<ParamField query="timeframe" type="string" default="30d">Time window</ParamField>
<ParamField query="model" type="string">AI model filter</ParamField>
<ParamField query="language" type="string">Language filter</ParamField>
<ParamField query="tags" type="string">Tag filter</ParamField>
<ParamField query="search" type="string">Search by domain or title</ParamField>
<ParamField query="sortBy" type="string" default="citations">Sort by: `citations`, `domain`, `type`</ParamField>
<ParamField query="page" type="integer" default="1">Page number</ParamField>
<ParamField query="limit" type="integer" default="50">Results per page</ParamField>

**Each source includes:** `id`, `url`, `domain`, `title`, `sourceType`, `urlType`, `citationCount`, `promptCount`, `averagePosition`
