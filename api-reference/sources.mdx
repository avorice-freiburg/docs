# Sources

## Source ranking

Returns sources most frequently cited by AI models when answering prompts in your industry.

<CodeGroup>

```bash cURL
curl -X GET "https://app.finseo.ai/api/v1/projects/{projectId}/sources?timeframe=30d&limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
response = requests.get(
    f"https://app.finseo.ai/api/v1/projects/{project_id}/sources",
    headers=headers,
    params={"timeframe": "30d", "limit": 10}
)
sources = response.json()["data"]["sources"]

for s in sources:
    print(f"{s['domain']} ({s['sourceType']}) — {s['citationCount']} citations")
```

</CodeGroup>

<ParamField query="timeframe" type="string" default="30d">Time window</ParamField>
<ParamField query="search" type="string">Search by domain or title</ParamField>
<ParamField query="sortBy" type="string" default="citations">`citations`, `domain`, `type`</ParamField>
<ParamField query="page" type="integer" default="1">Page number</ParamField>
<ParamField query="limit" type="integer" default="50">Results per page</ParamField>
