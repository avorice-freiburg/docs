# FinSEO API

The FinSEO API gives you programmatic access to your AI visibility data. Track how your brand appears across ChatGPT, Claude, Perplexity, and other AI platforms.

## What you can do

- **Retrieve visibility metrics** - Get daily KPIs and time series data for your brand's AI visibility
- **Track prompts** - List, add, and manage the AI prompts you're monitoring
- **Analyze competitors** - See how competitors rank in AI responses
- **Monitor sources** - Understand which sources AI models cite for your industry
- **Export data** - Bulk export all tracking data for your own analysis

## Base URL

```
https://app.finseo.ai/api/v1
```

## Quick start

```bash
# List your projects
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://app.finseo.ai/api/v1/projects

# Get visibility metrics for a project
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://app.finseo.ai/api/v1/projects/PROJECT_ID/metrics?timeframe=30d
```

## Response format

All responses follow a consistent envelope:

```json
// Success
{
  "data": { ... },
  "meta": { "page": 1, "limit": 50, "total": 142, "hasMore": true }
}

// Error
{
  "error": {
    "code": "NOT_FOUND",
    "message": "Project not found"
  }
}
```
